# pkmn-discovery

A web-based matching engine for identifying Pokémon based on a 9-dimensional trait search.

[View Live App](https://pkmn-discovery-kbxfm3bqwdorrdzmxzixfa.streamlit.app/)


This project utilizes data from [ratepkmn.com](https://ratepkmn.com/), a community-driven platform where over 12,000 contributors have scored Pokémon across 9 different attributes.
